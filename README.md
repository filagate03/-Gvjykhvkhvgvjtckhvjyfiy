# AI Таро-бот для Telegram

Этот проект представляет собой простого Telegram-бота, который использует AI для генерации раскладов Таро. Он написан на Node.js и использует библиотеку `telegraf` для взаимодействия с Telegram API и `openai` для работы с AI (через кастомный эндпоинт, совместимый с OpenAI).

## Инструкция по развертыванию на Railway

[Railway](https://railway.app/) — это платформа, которая позволяет легко и быстро развертывать приложения. Следуйте этим шагам, чтобы запустить вашего бота.

### Шаг 1: Подготовка

1.  **Аккаунт на GitHub:** Убедитесь, что у вас есть аккаунт на GitHub и код проекта находится в вашем репозитории.
2.  **Аккаунт на Railway:** Зарегистрируйтесь на [railway.app](https://railway.app/), если у вас еще нет аккаунта. Это можно сделать с помощью вашего GitHub-аккаунта.

### Шаг 2: Создание проекта на Railway

1.  Перейдите в ваш дашборд на Railway.
2.  Нажмите кнопку **"New Project"** (Новый проект).
3.  Выберите опцию **"Deploy from GitHub repo"** (Развернуть из GitHub-репозитория).
4.  Выберите репозиторий с кодом вашего бота.
5.  Railway автоматически определит, что это Node.js проект, и начнет процесс сборки и развертывания.

### Шаг 3: Настройка переменных окружения

Вашему боту нужны два секрета (API-ключа), чтобы он мог работать. Их нужно безопасно добавить в Railway.

1.  В дашборде вашего проекта на Railway перейдите на вкладку **"Variables"** (Переменные).
2.  Нажмите **"New Variable"** (Новая переменная) и добавьте две переменные:

    *   **`TELEGRAM_BOT_TOKEN`**:
        *   **Имя (Name):** `TELEGRAM_BOT_TOKEN`
        *   **Значение (Value):** `СЮДА_ВСТАВЬТЕ_ВАШ_ТЕЛЕГРАМ_ТОКЕН`

    *   **`GEMINI_API_KEY`**:
        *   **Имя (Name):** `GEMINI_API_KEY`
        *   **Значение (Value):** `СЮДА_ВСТАВЬТЕ_ВАШ_API_КЛЮЧ_GEMINI`

3.  После добавления этих переменных Railway автоматически перезапустит ваш проект (выполнит ре-деплой), чтобы применить новые настройки.

### Шаг 4: Готово!

Ваш бот должен быть успешно развернут и запущен. Вы можете проверить его статус и посмотреть логи на вкладке **"Deployments"** в вашем проекте на Railway.

Теперь вы можете найти своего бота в Telegram и отправить ему команду `/tarot`, чтобы получить свой первый расклад!
